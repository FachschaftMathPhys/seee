PEST Praktisches Evaluations ScripT ( >> FormPro) (Practical Evaluation ScripT)

Component: FIX

Looks for "failed choices" in the data output by the OMR component and allows the user to manually correct this. Corrected data is written in place, so once you close the script, no undo will be possible unless you re-recognize the sheets.

Known Bugs: When the preload is active, hitting enter seems to have no effect

Usage: fix.rb working_dir
