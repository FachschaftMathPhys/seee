Component: COMPARE

Compares two directories of YAML files against each other and compares each value. This allows checking for errors easily.

Usage: compare.rb working_dir compare_against_dir [fix]

Working Dir: Path to the files that need to be checked

Compare Dir: Path to the files that are absolutely correct

Fix: Specifiy if you want to open the FIX component for each difference. Goto the next difference by closing the FIX component
